
# Encode and Decode Strings

## Problem Statement

Design an algorithm to encode a list of strings into a single string. The encoded string should then be decoded back to the original list of strings.

## Implementation

### Example 1:
#### Input:
```plaintext
["neet", "code", "love", "you"]
```
#### Output:
```plaintext
["neet", "code", "love", "you"]
```

### Example 2:
#### Input:
```plaintext
["we", "say", ":", "yes"]
```
#### Output:
```plaintext
["we", "say", ":", "yes"]
```
